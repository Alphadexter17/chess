PICT B.E. Course Structure

First Year (FE)

Engineering Mathematics-I: Calculus, Linear Algebra basics.

Basic Electrical Engineering: Circuits, AC/DC fundamentals.

Second Year (SE)

Data Structures: Arrays, Linked Lists, Trees.

Computer Networks: OSI Model, TCP/IP protocols.

Third Year (TE)

Database Management Systems: ER models, SQL, Normalization.

Operating Systems: Process management, Scheduling, Memory management.

Final Year (BE)

Software Engineering: SDLC models, Project management.

Machine Learning: Supervised and Unsupervised learning algorithms.
